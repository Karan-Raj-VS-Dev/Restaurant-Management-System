# operations-insights-service

## Purpose

`operations-insights-service` provides cross-service analytics for order flow, table usage, server performance, and stock intelligence.

## Default Port

- `8097`

## Current APIs

- `GET /api/insights/daily`
- `GET /api/insights/stock`

## Current Scope

- daily operations summary placeholder
- stock health insight placeholder

## Planned Responsibilities

- read-only analytics projections
- busiest table and top server calculations
- current stock snapshot views
- low-stock and out-of-stock analytics
- ingredient consumption and menu impact analysis
- input to management dashboards
