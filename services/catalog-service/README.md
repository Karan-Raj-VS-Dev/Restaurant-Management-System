# catalog-service

## Purpose

`catalog-service` owns menu items, pricing, and recipe definitions for the restaurant.

## Default Port

- `8086`

## Current APIs

- `GET /api/menu/items`

## Current Scope

- menu item list
- price placeholders
- recipe ingredient placeholders
- sample availability flag at item level

## Planned Responsibilities

- menu persistence
- category and modifier management
- recipe management
- property-specific pricing
- menu publication windows
- integration with stock-aware availability projections
