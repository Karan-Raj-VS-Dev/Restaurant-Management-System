# reporting-service

## Purpose

`reporting-service` exposes summarized operational and financial reporting views.

## Default Port

- `8096`

## Current APIs

- `GET /api/reports/daily-summary`

## Current Scope

- sample daily summary endpoint
- placeholder metrics for orders, sales, takeaway, and kitchen backlog

## Planned Responsibilities

- scheduled reports
- property-level and cross-property reporting
- export-friendly summaries
- report filtering by date, outlet, and channel
- management dashboards and downloadable reports
